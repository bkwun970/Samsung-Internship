### Javascript 
- 따옴표

    1. 큰따옴표: "Hello"
    2. 작은따옴표: 'Hello'
    3. 역 따옴표(백틱, backtick): ``
    
```    
    let name = "John";

// 변수를 문자열 중간에 삽입
alert( `Hello, ${name}!` ); // Hello, John!

// 표현식을 문자열 중간에 삽입
alert( `the result is ${1 + 2}` ); // the result is 3
```
